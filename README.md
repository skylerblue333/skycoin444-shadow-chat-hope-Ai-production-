- **Express 4** - Fast, minimalist web framework
- **tRPC 11** - End-to-end type safety
- **Drizzle ORM** - Type-safe database queries (MySQL)
- **MySQL** - Relational database with 38 tables
- **Node.js v22** - JavaScript runtime
- **SuperJSON** - Serialization with Date/Map support
- **tsx** - TypeScript execution

- **Manus WebDev** - Autoscale hosting with custom domains

- **OAuth 2.0** - Manus authentication
- **Stripe** - Payment processing (test sandbox ready)
- **WebSocket** - Real-time notifications
- **Heartbeat** - Background job scheduling

## 🛠️ Development

### Prerequisites
- Node.js v22+
- pnpm v10+
- MySQL 8.0+

### Installation
```bash
# Clone repository
git clone https://github.com/skylerblue333/skycoin4444.git
cd skycoin_production

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local

# Run database migrations
pnpm db:push

# Start development server
pnpm dev
```

### Available Scripts
```bash
pnpm dev          # Start dev server (http://localhost:3000)
pnpm build        # Production build
pnpm start        # Start production server
pnpm test         # Run tests (vitest)
pnpm check        # TypeScript type checking
pnpm format       # Format code (prettier)
pnpm db:push      # Apply database migrations
```
