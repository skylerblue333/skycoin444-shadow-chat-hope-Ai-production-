## 🚀 Getting Started

### Quick Start (5 Minutes)

```bash
# 1. Clone repository
git clone https://github.com/skylerblue333/skycoin4444.git
cd skycoin4444

# 2. Install dependencies
pnpm install

# 3. Configure environment
cp .env.example .env.local
# Edit .env.local with your settings

# 4. Start development server
pnpm dev

# 5. Open browser
open http://localhost:3000
```

## 🐛 Known Issues & Limitations

### Beta Limitations

1. **TypeScript Errors** - 1591 TS errors in generated files (non-blocking)
   - Impact: Development experience, not production
   - Fix: Planned for v1.0.0 final release

2. **Mining Pools** - Real pools configured but may need adjustment
   - Impact: Mining efficiency varies by pool
   - Fix: Auto-optimization in v1.0.1

3. **Dating Matching** - Algorithm in beta phase
   - Impact: Match quality improves with more users
   - Fix: ML model improvements in v1.0.1

4. **Payment Processing** - Stripe test mode only
   - Impact: Real transactions not processed
   - Fix: Production mode in v1.0.0 final

### Performance Notes

- Initial load may take 5-10 seconds on first visit
- Mining system takes 1-2 minutes to reach full capacity
- Database queries may be slow with large datasets (1M+ records)
